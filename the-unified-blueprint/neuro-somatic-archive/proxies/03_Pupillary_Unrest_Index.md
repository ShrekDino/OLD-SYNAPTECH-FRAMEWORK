# 3. Pupillary Unrest Index (PUI)

**Physiological Substrate:** Spontaneous pupillary fluctuation in darkness reflects locus coeruleus-norepinephrine (LC-NE) tonic firing mode.

**Administration Protocol:**
1. Dark adaptation 2 minutes
2. Observe pupil through penlight dim beam for 60 seconds
3. Count visible fluctuations (oscillations in diameter)
4. Compute: PUI = fluctuations per minute

**Pathological Thresholds:**
| PUI | Interpretation |
|---|---|
| <4/min | Normal — phasic LC firing mode |
| 4–8/min | Hyperarousal — mixed LC mode |
| 8–12/min | Significant hyperarousal |
| >12/min | Severe — tonic LC firing mode |

**Phenotype Cross-Reference:** PTSD (hypervigilance), Bipolar (manic), Panic Disorder, Generalized Anxiety, Stimulant Intoxication, Stimulant Withdrawal, Insomnia, Nightmare Disorder.
