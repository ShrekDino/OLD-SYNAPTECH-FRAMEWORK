# 5. Prepulse Inhibition (PPI) — Clinical Variant

**Physiological Substrate:** Sensorimotor gating mediated by hippocampal-striatal-pallidal-thalamic circuitry. Prepulse suppresses startle via inhibitory interneuron activation.

**Administration Protocol:**
1. Deliver weak prepulse stimulus (finger snap at 30 cm, ≈70 dB) 120 ms before main startle stimulus
2. Deliver handclap (95 dB) alone — note startle magnitude (control)
3. Deliver prepulse + handclap pair — note suppressed startle (test)
4. Compute: PPI = (1 − test/control) × 100%

**Pathological Thresholds:**
| PPI | Interpretation |
|---|---|
| >50% inhibition | Normal sensorimotor gating |
| 30–50% | Borderline |
| <30% | Significant gating deficit |

**Phenotype Cross-Reference:** Schizophrenia, Autism Spectrum, PTSD, OCD, Tourette's, Tic Disorders, Schizotypal Personality, Bipolar (manic).
