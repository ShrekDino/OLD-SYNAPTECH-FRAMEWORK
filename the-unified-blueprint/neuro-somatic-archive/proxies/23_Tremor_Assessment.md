# 23. Tremor Assessment (Visual + Paper Task)

**Physiological Substrate:** Basal ganglia-thalamo-cortical oscillatory loop vs. enhanced physiological tremor vs. cerebellar intention tremor.

**Administration Protocol:**
1. Resting: hands in lap, observe for pill-rolling tremor
2. Postural: arms extended, fingers spread, hold 30s
3. Kinetic: finger-to-nose test
4. Writing: patient writes a sentence; draw a spiral

**Pathological Thresholds:**
| Type | Normal | Pathological |
|---|---|---|
| Resting tremor | Absent | Present = Parkinsonism, antipsychotic-induced |
| Postural tremor | Fine 8–12 Hz | Coarse = enhanced physiological, anxiety, stimulant, alcohol withdrawal, hyperthyroid |
| Intention tremor | Absent | Present = cerebellar pathology |
| Writing tremor | Smooth | Tremulous = essential tremor |

**Phenotype Cross-Reference:** Parkinson's Disease, Antipsychotic-Induced Parkinsonism, Essential Tremor, Alcohol Withdrawal, Stimulant Intoxication, Anxiety Disorders, Hyperthyroidism, Cerebellar Neurocognitive, Tardive Dyskinesia.
