# 15. Digit Span (Forward + Backward)

**Physiological Substrate:** Dorsolateral PFC working memory buffer — phonological loop (forward) + central executive (backward).

**Administration Protocol:**
1. FORWARD: Read digits at 1/s, patient repeats in same order. Start at 3 digits, increase until failure. Record max span.
2. BACKWARD: Read digits at 1/s, patient repeats reversed. Start at 2 digits, increase. Record max span.

**Pathological Thresholds:**
| Measure | Normal | Pathological |
|---|---|---|
| Forward span | 6–7 digits | <5 = attentional deficit |
| Backward span | 4–5 digits | <3 = executive dysfunction |
| Difference forward-backward | 2–3 | >3 or <1 = frontal-subcortical dysconnectivity |

**Phenotype Cross-Reference:** Attention-Deficit/Hyperactivity, Schizophrenia, Major Depressive, Neurocognitive Disorders (all), Delirium, Anxiety (attentional interference), PTSD (concentration deficit).
