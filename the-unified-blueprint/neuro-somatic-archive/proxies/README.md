# Zero-Cost Forensic Proxy Protocols

This reference compiles the complete set of thirty (30) zero-cost forensic proxy markers from the DSM-6 Practitioner's Field Guide. These are manually administrable clinical assessments requiring no equipment beyond a timing device and penlight (stethoscope optional). Each protocol describes the physiological substrate, step-by-step administration procedure, pathological thresholds, and phenotype cross-reference for forensic and clinical application.

## Anatomy of a Proxy Examination (Standard Field Protocol)

| Step | Protocol | Time | Notes |
|------|----------|------|-------|
| 1 | Vagal Tone (RSA) | 3 min | Radial pulse RSA — first priority; informs all subsequent interpretation |
| 2 | Pupillary Reflex | 2 min | Baseline and constriction velocity → autonomic set point |
| 3 | Breath-Hold | 1 min | Chemoreflex integrity |
| 4 | Startle | 3 min | 3 trials, assess habituation → brainstem sensitization |
| 5 | HR Reactivity | 3 min | Orthostatic or script-driven → autonomic response capacity |
| 6 | PPI | 2 min | If sensorimotor gating is in question |
| 7 | Cognitive Screening | 5 min | Clock, 3-word recall, animal fluency, digit span → if NCD suspected |
| 8 | Motor Battery | 5 min | DTR, gait, tremor, rapid alternating movements → if motor circuit pathology suspected |

**Total examination time:** 24 minutes (complete) or 9 minutes (rapid assessment: steps 1–5)
