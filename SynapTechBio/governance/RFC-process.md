# RFC Process (Deferred)

> *Status: DESIGN COMPLETE — NOT YET ACTIVE. Deferred until the team reaches 6+ FTE.*

## Current Pre-Seed Policy

Major decisions are made by the founder. Community feedback is collected via GitHub issues and Discord discussions. No formal RFC process is active.

## Deferred Structure (Post-Seed, 6+ FTE)

1. **Proposal** — GitHub issue tagged `RFC`
2. **Discussion** — 2-week community comment period
3. **Refinement** — Author revises based on feedback
4. **Decision** — Community Council + founder consensus
5. **Implementation** — Changes merged
