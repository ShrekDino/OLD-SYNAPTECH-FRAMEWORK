# How You Can Help

This monorepo is the **single entry point** for the entire SynapTechBio ecosystem. If you want to contribute, this is where to start.

## For Neuroscientists

- Validate the connectome-to-computation pipeline in `packages/idre/`
- Propose research collaborations using the FlyWire LSM (`packages/flywire-lsm/`)
- Review the CSDF thermodynamic framework (`packages/csdf/`)

## For Engineers

- **IDRE** (`packages/idre/`): GPU-accelerated sparse graph, FastAPI backend, React/Three.js frontend
- **FlyWire LSM** (`packages/flywire-lsm/`): Reservoir computing, ridge regression, SSE streaming
- **FlyWire Realtime** (`packages/flywire-realtime/`): Closed-loop simulation, body physics, 60 Hz visualization
- **Cosmos** (`packages/cosmos/`): World model integration, video tokenization, physical reasoning

## For Grant Writers

We're seeking STTR/SBIR grant partners. The monorepo provides a single source of truth for our technical capability.
